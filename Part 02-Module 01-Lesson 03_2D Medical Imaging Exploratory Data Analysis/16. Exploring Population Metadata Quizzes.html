16. Exploring Population Metadata Quizzes

Exploring Population Metadata Quizzes

Exploring population metadata

Which of the following could you NOT assess if you had only the following information about your population:

  • Average age of the whole population
  • Each patient's gender
  • Each patient's race
  • Count of the number of DICOM series for each patient
SOLUTION: Relationship between age and number of DICOM series

Data visualizations

Which type of data visualization tool would you use to see how often members of your population have both cancer and hypertension, compared to members who have both cancer and diabetes?

SOLUTION: Co-occurrence matrix
